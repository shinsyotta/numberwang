Numberwang
==========

http://www.youtube.com/watch?v=qjOZtWZ56lc

Installation
============

```ruby
gem install numberwang

require "numberwang"
```

OR... In Rails, just insert this into your Gemfile:

```ruby
gem "numberwang"
```

Use
===

```ruby
1.numberwang?

=> true

1.7.numberwang?

=> false
```

etc.
